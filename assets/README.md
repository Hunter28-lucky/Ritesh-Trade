# Assets Folder

This folder is for storing website assets like:

- **favicon.ico** - Website favicon (browser tab icon)
- **logo.png** - Company logo
- **images/** - Additional images
- **videos/** - Video content
- **documents/** - Downloadable PDFs, etc.

## Adding Your Logo

1. Save your logo file as `logo.png` or `logo.svg`
2. Update the reference in `index.html`

## Adding a Favicon

1. Create a favicon (you can use https://favicon.io)
2. Save as `favicon.ico` in this folder
3. The HTML already references it!

## Image Optimization Tips

- Use WebP format for better compression
- Resize images to appropriate dimensions
- Use lazy loading (already implemented in HTML)
- Compress images before uploading

## Recommended Tools

- **TinyPNG** - Compress images
- **Favicon.io** - Generate favicons
- **Unsplash** - Free stock photos
- **Canva** - Design graphics
