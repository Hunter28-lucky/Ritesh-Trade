üé® Customization Guide

1. Change Colors

Open css/styles.css and modify these variables at the top:

:root {
    --primary-color: #3382D1;      /* Main blue color */
    --secondary-color: #66BAF1;    /* Light blue */
    --dark-color: #151525;         /* Dark navy */
    --light-color: #F5F9FF;        /* Off-white */
}

2. Update Content

Open index.html and edit the text directly. Each section is clearly labeled:

  • <section id="home"> - Hero section
  • <section id="about"> - About section
  • <section id="services"> - Services section
  • <section id="reviews"> - Reviews section
  • <section id="contact"> - Contact section

3. Change Images

Find image tags in index.html and replace the src attribute:

<img src="YOUR_IMAGE_URL_HERE" alt="Description">

4. Add Your Logo

  1. Save your logo as favicon.ico in the assets/ folder
  2. Update the header logo in index.html

üîß Troubleshooting

Website not loading?

  • Make sure all files are in the correct folders
  • Check that file names match exactly (case-sensitive)
  • Try opening in a different browser

Styles not showing?

  • Verify css/styles.css exists
  • Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)
  • Check browser console for errors (F12)

JavaScript not working?

  • Verify js/script.js exists
  • Check browser console for errors (F12)
  • Make sure JavaScript is enabled in your browser
